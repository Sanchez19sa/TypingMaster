# 🎯 TypeMaster

Una aplicación web moderna para practicar mecanografía con soporte para múltiples modos (texto, código) y análisis avanzado de rendimiento con gráficas en tiempo real.


## 🚀 Features

- ✅ **Múltiples Modos:** Texto normal, código (Python, JavaScript, TypeScript)
- 📊 **Análisis Avanzado:** Gráficas de WPM, precisión y consistencia con Recharts
- 🌐 **Multilenguaje:** Soporte para Español e Inglés
- ⚙️ **Personalizable:** Duración (15s-120s) y dificultad ajustable
- 📈 **Estadísticas Detalladas:** WPM, precisión, palabras correctas/incorrectas
- 🎨 **UI Moderna:** Diseño limpio y responsivo

## 🛠️ Tech Stack

- **Frontend:** React 18 + TypeScript
- **Charts:** Recharts
- **Styling:** CSS Modules
- **State Management:** React Hooks (useState, useCallback, useRef)
- **i18n:** Sistema de traducciones personalizado
- **Build Tool:** Vite
